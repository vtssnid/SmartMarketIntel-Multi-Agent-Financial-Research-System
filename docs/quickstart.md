# Quick Start

1. Open `notebook/AAI_520_Final_Team_Project_Enhanced.ipynb` in Google Colab.
2. Install dependencies using `requirements.txt`.
3. Set any required API keys via environment variables in Colab.
4. Run the orchestrator cell to build the `bundle`.
5. Run the enhancement sections to generate visualizations and evaluation feedback.
6. Export charts to `img/` and include them in the README and design document.
